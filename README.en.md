A small Windows desktop app for shrinking images to **HD/1080p** – with an
optional **capture-date overlay**.

![App screenshot](docs/screenshot.png)

## Features

- 📂 **Import / select images** (multi-selection)
- 📅 Optionally burn the **capture date** (from EXIF) into the image
  - choose the **corner** (top/bottom, left/right)
  - choose the **text color**
- 💾 Freely selectable **output folder**
- 🖼️ Output in **HD quality** (max. 1920 × 1080, aspect ratio preserved)
- 🎚️ Quality defaults to **80** (≈ max. 20 % quality loss) and can be set
  **manually** (1–100)

## Running

### Option A – with Python installed
Requires Python 3.10+ and [Pillow](https://pypi.org/project/pillow/).

```bash
pip install pillow
python magic_pictures.py
```

Or double-click **`Magic-Pictures starten.bat`**.

### Option B – portable EXE (no Python)
Double-clicking **`Portable-EXE bauen.bat`** builds a standalone
`dist\Magic-Pictures.exe` that runs without an installed Python (e.g. from a
USB stick). A pre-built version is also available on the
[Releases page](../../releases).

## Customizing the icon

The app icon is generated by `icon_erstellen.py`:

```bash
python icon_erstellen.py
```

## Project files

| File | Purpose |
|------|---------|
| `magic_pictures.py` | The app (Tkinter UI + image processing) |

| `Magic-Pictures starten.bat` | Double-click launcher (requires Python) |
